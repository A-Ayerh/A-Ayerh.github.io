Spatial Mapping with the Brain

Andrew Ayerh

Hi, I'm Andrew Ayerh, the creator of this project. My journey through neuroscience, machine learning, and neural interfacing has inspired me to create a spatial mapping system using real-time EEG data.

Leveraging my experience at EMOTIV and Freelance Robotics, I’ve developed this project to explore how spatial mapping can be used to understand brain activity during motor tasks and other cognitive functions.

This GitHub Pages site is new, and many sections are still under development. I’ll be adding more content and demos soon, so stay tuned!
